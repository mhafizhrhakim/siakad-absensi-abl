SMARTPRESENCE - Prototype Absensi QR Kampus

Isi prototype:
1. Login Mahasiswa
2. Register Mahasiswa via NIM
3. Login Dosen
4. Register Dosen via NIDN/NIK
5. Dashboard Mahasiswa
6. Jadwal Kuliah Mahasiswa
7. Scan QR Kehadiran Mahasiswa
8. Konfirmasi Kehadiran
9. Riwayat Kehadiran
10. Rekap Semester Mahasiswa
11. Detail Mata Kuliah Mahasiswa
12. Notifikasi Mahasiswa
13. Profil Mahasiswa
14. Dashboard Dosen
15. Jadwal Mengajar Dosen
16. Generate QR Absensi Dosen
17. Monitoring Kehadiran Real-time
18. Daftar Kehadiran
19. Rekap Semester Dosen
20. Profil Dosen

Catatan:
- Ini prototype HTML, CSS, JS.
- Scan QR memakai simulasi kamera browser.
- Ada fallback input kode manual jika kamera tidak support.
- Untuk production, scanner sebaiknya memakai library seperti html5-qrcode atau BarcodeDetector API, backend auth, database, dan QR token dinamis.
